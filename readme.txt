Add CrunchBase company details widgets to your posts , pages and blog widgets

== Description ==

Easily include CrunchBase widgets, example syntax:

* &#91;cb type="company"&#93;iTeam Web Solutions&#91;/cb&#93;

The plugin is made for companies only. But surely update soon for all other services

Please create support tickets for the new functionality  that you need, i will update ASAP.

== Installation ==

1. Upload CrunchBase Widget folder to the `/wp-content/plugins/` directory or download directly from plugins page
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets page
4. Include a widget in a post or page by adding the appropriate shortcode to your page or post.
5. Set the API key - Register for API key from http://developer.crunchbase.com/ and input that key in the settings
